The Flash: The Running Game(Prototype1):

Installation:
	  	-  Lancer le menu : $python TFTRGP1.  
		-  Lancer le jeu: Appuyez sur Entrée pour passer les deux introductions.  
Jouer :
		-D, espace, pour lancer une attaque ou sauter. 
		
		-Fin de partie: Taper votre nom de joueur, Puis Enter.
		
		-Appuyer de nouveau sur entrée pour commencer directement une nouvelle partie:

Regles : 
		- Survive le plus longtemps possible
		- Une attaque dans la tête du Reverse Flash et c'est +20Pts Bonus !
